Настоящее соглашение регулирует использование сайта и программного обеспечения Avenor Downloader.

1. Описание сервиса

Avenor Downloader — программное обеспечение для загрузки, конвертации и обработки мультимедийного контента.

2. Подписки и лицензия

Сервис предоставляется по подписке: Free, PRO Monthly, PRO Yearly, PRO Studio.

Подписка не является покупкой программного обеспечения и действует только в период активной оплаты.

3. Устройства

Каждая подписка имеет ограничение по количеству устройств. При превышении лимита доступ может быть временно ограничен.

  • качество загрузки;
  • лимиты и безлимиты по тарифу;
  • пакетная загрузка;
  • коммерческое использование.

5. Коммерческое использование

Коммерческое использование разрешено только в тарифе PRO Studio.

Avenor Downloader не передаёт авторские права на контент.

6. Платежи

Все платежи обрабатываются системой Paddle (Merchant of Record). Подписка может автоматически продлеваться.

Сервис предоставляется «как есть». Avenor Downloader не несёт ответственности за блокировки, изменения API платформ или невозможность загрузки контента.
